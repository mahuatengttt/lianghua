# 量子 - A股量化交易系统需求文档

## 一、A股主流量化交易策略分类

### 1. 趋势跟踪策略 (Trend Following)
- **原理**：顺势而为，买入上涨趋势、卖出下跌趋势
- **核心指标**：移动平均线(MA)、MACD、布林带、ADX、抛物线SAR
- **经典策略**：
  - 双均线交叉：快线上穿慢线做多，下穿做空
  - 海龟交易法则：唐奇安通道突破入场，ATR动态止损
  - 动量策略：N日收益率排序，买入高动量卖出低动量
- **A股适配**：结合成交量确认、避开震荡市

### 2. 均值回归策略 (Mean Reversion)
- **原理**：价格围绕均值波动，偏离过大时反向交易
- **核心指标**：RSI、布林带宽度、Z-Score、CCI
- **经典策略**：
  - 布林带反转：触碰上下轨反向开仓
  - RSI极端值：RSI<20超买反弹、RSI>80超卖回落
  - 统计套利配对：价差Z-Score回归均值开仓
- **A股适配**：适用于震荡行情、流动性好的股票

### 3. 套利策略 (Arbitrage)
- **原理**：利用相关资产间的定价偏差获利
- **子类别**：
  - **统计套利/配对交易**：同行业两只股票价差回归
  - **ETF套利**：ETF价格与净值偏差
  - **期现套利**：股指期货与现货基差
  - **跨期套利**：同一品种不同到期月份
- **A股适配**：融资融券标的、ETF申赎机制

### 4. 因子策略 / 多因子模型
- **原理**：通过多个因子综合评分选股
- **核心因子**：
  - 估值因子：PE、PB、PS、PCF
  - 成长因子：营收增长、净利润增长
  - 质量因子：ROE、ROA、毛利率
  - 动量因子：过去3-12个月收益
  - 波动率因子：低波动异象
  - 情绪因子：换手率、资金流向
- **框架**：Barra模型、Fama-French多因子模型

### 5. 机器学习预测策略 (ML/DL)
- **原理**：用历史数据训练预测模型
- **常用模型**：
  - 集成学习：XGBoost、LightGBM、Random Forest
  - 深度学习：LSTM、GRU、Transformer、TFT
  - 强化学习：DQN、PPO、SAC（FinRL框架）
- **特征工程**：量价技术指标 + 宏观数据 + 舆情因子
- **A股适配**：需处理过拟合、样本外衰减问题

### 6. 事件驱动策略
- **原理**：利用特定事件驱动的短期价格异动
- **事件类型**：财报超预期、分红送转、重组公告、产业政策

### 7. 高频交易策略
- **原理**：利用速度优势获取微小价差
- **类型**：做市商策略、订单流分析、盘口博弈

## 二、系统架构设计

### 核心设计原则
1. **事件驱动**：所有模块通过事件总线通信
2. **松耦合**：模块间仅通过接口协议交互
3. **可扩展**：策略/数据源/券商均通过插件化方式接入
4. **回测-实盘一致**：同一策略代码无缝切换

### 五大模块职责
| 模块 | 职责 | 核心类 |
|------|------|--------|
| Data | 数据获取、清洗、存储 | DataSource, DataStore, Processor |
| Strategy | 信号生成、组合构建 | BaseStrategy, PortfolioBuilder |
| Backtest | 历史回测、绩效分析 | BacktestEngine, Analyzer, Report |
| Risk | 风控、仓位管理 | RiskManager, PositionManager, CircuitBreaker |
| Broker | 交易执行、账户管理 | BaseBroker, OrderManager |
| Monitor | 系统监控、日志告警 | MetricsCollector, AlertManager, Dashboard |

### 策略研发流水线
```
数据获取 → 数据预处理 → 特征工程 → 策略回测 → 绩效分析 → 参数优化 → 实盘部署
```

## 三、重要技术要点

### 回测引擎设计
- **Tick驱动**：逐笔回放历史数据进行撮合
- **Bar驱动**：以K线为单位模拟交易
- **订单类型**：市价单、限价单、止损单、条件单
- **费用模拟**：佣金+印花税+滑点
- **涨跌停处理**：A股特有机制

### A股特殊机制
- T+1 交易规则
- 涨跌停板限制(±10%/±20%)
- 集合竞价(9:15-9:25)与连续竞价
- 融资融券门槛
- 停复牌处理
- 分红除权除息

### 风险控制体系
1. 单品种仓位上限
2. 最大回撤控制
3. Value-at-Risk (VaR)
4. 波动率敞口限制
5. 相关性风险监控
6. 流动性风险评估
7. 熔断机制
